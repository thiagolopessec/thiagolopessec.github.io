# NIST CSF Mapping

## Identify (ID)
- Inventory assets (hosts, vendors) and local attack surface (open ports).

## Protect (PR)
- Reduce exposure (router management hardening, IoT segmentation).

## Detect (DE)
- Add monitoring and baseline alerting (Wazuh/Suricata).

## Respond (RS)
- Define an incident SOP: isolate device, rotate credentials, update firmware, review logs/traffic.

## Recover (RC)
- Backups of router config, topology documentation, simple restoration checklist.
